### lilyterm
需要 libvte-dev，github 上的版本較新
```
git clone https://github.com/Tetralet/LilyTerm.git
cd LilyTerm
./configure --prefix=/usr --sysconfdir=/etc/xdg
make
sudo make install
```
參考：http://lilyterm.luna.com.tw/ 及 https://github.com/Tetralet/LilyTerm
