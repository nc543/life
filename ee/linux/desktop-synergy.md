# synergy -- 多台電腦鍵鼠共用

安裝：`sudo apt-get install synergy`

「開始選單」→「附屬應用程式」→「Synergy」有 GUI 可以設定

設定檔：~/.config/Synergy/Synergy.conf

Put <code>screenSaverSync</code> to <code>false</code>
